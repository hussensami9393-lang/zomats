# بوت تيليجرام لبيع السلع مقابل رصيد آسياسيل

هذا بوت متجر بسيط لتليجرام يسمح لك كمدير بإضافة السلع، عرضها للزبائن، استقبال طلبات الشراء، واستلام ملاحظات الدفع اليدوي عبر رصيد آسياسيل. العميل يختار السلعة، ثم تظهر له تعليمات الدفع، وبعد أن يرسل رقم الهاتف أو إثبات التحويل يصل الطلب للمدير للموافقة أو الرفض.

## المميزات

- عرض السلع للزبائن بأزرار داخل تيليجرام.
- إضافة سلع من داخل البوت بواسطة المدير.
- حفظ المنتجات والطلبات في قاعدة بيانات SQLite.
- دفع يدوي عبر رصيد آسياسيل.
- إرسال الطلب الجديد للمدير مباشرة مع أزرار قبول أو رفض.
- إشعار العميل عند قبول أو رفض الطلب.
- تقليل المخزون تلقائيًا عند قبول الطلب.
- دعم أكثر من مدير عبر `ADMIN_IDS`.

## المتطلبات

- Python 3.10 أو أحدث.
- توكن بوت تيليجرام من BotFather.
- آيدي حساب المدير في تيليجرام.

## إنشاء بوت من BotFather

افتح تيليجرام وابحث عن:

```text
@BotFather
```

ثم أرسل:

```text
/newbot
```

اختر اسم البوت واسم المستخدم، ثم انسخ التوكن.

## معرفة آيدي المدير

يمكنك معرفة آيدي حسابك بإرسال رسالة إلى بوت مثل:

```text
@userinfobot
```

انسخ الرقم وضعه في ملف `.env` داخل `ADMIN_IDS`.

## التثبيت

ادخل إلى مجلد المشروع:

```bash
cd asiacell_shop_bot
```

أنشئ بيئة افتراضية:

```bash
python -m venv .venv
```

فعّل البيئة على Linux أو macOS:

```bash
source .venv/bin/activate
```

على Windows:

```bat
.venv\Scripts\activate
```

ثبّت المتطلبات:

```bash
pip install -r requirements.txt
```

انسخ ملف الإعدادات:

```bash
cp .env.example .env
```

افتح ملف `.env` وعدّل القيم:

```env
BOT_TOKEN=ضع_توكن_البوت_هنا
ADMIN_IDS=ضع_ايدي_حسابك_هنا
SHOP_NAME=متجر السلع
DB_PATH=data/shop.db
PAYMENT_INSTRUCTIONS=حوّل رصيد آسياسيل إلى الرقم: 07XXXXXXXXX ثم أرسل رقم الهاتف أو إثبات التحويل هنا.
```

إذا كان لديك أكثر من مدير:

```env
ADMIN_IDS=111111111,222222222
```

## التشغيل

شغّل البوت:

```bash
python bot.py
```

افتح البوت في تيليجرام وأرسل:

```text
/start
```

إذا كان حسابك ضمن `ADMIN_IDS` سيظهر زر لوحة المدير، ويمكنك أيضًا فتحها بالأمر:

```text
/admin
```

## إضافة سلعة

من داخل البوت:

1. اضغط لوحة المدير.
2. اضغط إضافة سلعة.
3. أرسل اسم السلعة.
4. أرسل الوصف.
5. أرسل السعر، مثل: `5000 دينار` أو `10000 رصيد آسياسيل`.
6. أرسل كمية المخزون كرقم.

بعد ذلك ستظهر السلعة للعملاء في قائمة السلع.

## طريقة شراء العميل

1. العميل يضغط عرض السلع.
2. يختار السلعة.
3. يضغط شراء.
4. تظهر له تعليمات الدفع عبر آسياسيل.
5. يرسل رقم الهاتف أو إثبات التحويل.
6. يصل الطلب للمدير.
7. المدير يضغط قبول أو رفض.
8. يصل إشعار للعميل بنتيجة الطلب.

## ملاحظات مهمة

هذا البوت لا يتحقق تلقائيًا من تحويل رصيد آسياسيل، بل يعتمد على المراجعة اليدوية من المدير. إذا توفرت لديك لاحقًا بوابة دفع رسمية أو API قانوني للتحقق من الدفع، يمكن تطوير البوت ليربط بها.

لا تضع توكن البوت أو بيانات حساسة داخل المحادثات العامة. احتفظ بملف `.env` على جهازك أو سيرفرك فقط.

## تشغيل دائم على سيرفر

يمكنك استخدام `tmux`:

```bash
tmux new -s shopbot
source .venv/bin/activate
python bot.py
```

للخروج وتركه يعمل:

```text
Ctrl + B ثم D
```

للرجوع:

```bash
tmux attach -t shopbot
```
